The main navigation uses TWO primary entry points reflecting the platform's mental model:

1. **Home** (single link)
2. **Get Settled** (dropdown — Canada Connect / utility layer): Find Services, Events, How-To Guides, City Guides, Newcomer Checklists, For Businesses
3. **Explore the City** (dropdown — FirstBitesTO / discovery layer): Food Experiences (/restaurants), Toronto Food Trails (external taste-to-trails)
4. **Admin** (conditional, admins only)

Help Centre lives in the top utility bar. Login/List Your Business sit in the right-side action area. Mobile menu mirrors the same two-section grouping with "Get Settled" and "Explore the City" headers. Never use language like "directory", "listing", or "category" inside FirstBitesTO surfaces — use "experience", "trail", "guide", "discovery".
